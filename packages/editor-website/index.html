Toolbar configuratie

De toolbar kan geconfigureerd worden via het toolbar-config attribuut. Dit attribuut accepteert een JSON-array van groepen, waarbij elke groep een id en een lijst van items heeft. Optioneel kan een label worden meegegeven als aria-label voor de groep.

Beschikbare items

format-select, language-select, bold, italic, underline, strike, code, highlight, superscript, subscript, undo, redo, ordered-list, bullet-list, definition-list, insert-table, text-align, link, image-upload, horizontal-rule, text-direction-ltr, text-direction-rtl, keyboard-shortcuts, accessibility-notifications

Voorbeeld: minimale toolbar

Hieronder staat een editor met een minimale toolbar configuratie: alleen tekststijlen en een link-knop.

Deze editor heeft een minimale toolbar.

Voorbeeld HTML

<clippy-editor
  toolbar-config='[
    { "id": "text-styling", "items": ["bold", "italic", "underline"] },
    { "id": "history", "items": ["undo", "redo"] },
    { "id": "insert", "items": ["link"] }
  ]'
>
  <div slot="content">
    <p>Inhoud hier...</p>
  </div>
</clippy-editor>

Voorbeeld: groepen met aangepaste labels

Gebruik het label veld om een eigen aria-label voor een groep in te stellen. 

<clippy-editor
  toolbar-config='[
    { "id": "opmaak", "label": "Tekstopmaak", "items": ["bold", "italic", "code"] },
    { "id": "invoegen", "label": "Invoegen", "items": ["link", "image-upload", "insert-table"] }
  ]'
>
  ...
</clippy-editor>
